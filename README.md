# 任务型多轮对话管理系统 —— 后端
## 系统功能
- 面向领域应用的多轮问答服务生成系统
- 用户可基于业务场景自定义配置
- 提供了一种脚本语言来描述复杂对话逻辑
- 可选择的内置深度学习算法
- 生成符合领域业务需求的对话机器人
- 前端地址 [here](https://github.com/tao2years/DSWeb2)
